## 1. 먼저, 마케팅이 뭔가요? (5분 개념 정리)

마케팅은 어렵게 생각할 필요 없습니다. 딱 4가지 질문에 답하는 일입니다.

| 질문 | 마케팅 용어 | 이 문서에서 다루는 곳 |
|---|---|---|
| **누구에게** 팔 건가? | 타깃 / 페르소나 | 3장 |
| **무엇을** 말할 건가? | 메시지 / 포지셔닝 | 4장 |
| **어디서** 알릴 건가? | 채널 | 5장 |
| **얼마나** 잘 됐나? | KPI / 성과 측정 | 9장 |

> 💡 **핵심 원칙 한 줄:** 좋은 마케팅은 "우리 제품 좋아요!"라고 외치는 게 아니라,
> **고객의 고민을 대신 말해주고 → 해결책을 보여주는 것**입니다.

**고객 여정(퍼널)** 이라는 개념도 알아두면 좋습니다. 고객은 이 순서로 움직입니다.

```
① 인지(알게 됨) → ② 관심(궁금해함) → ③ 설치/가입 → ④ 사용(활성) → ⑤ 결제(프리미엄/구매) → ⑥ 재구매/추천
```

우리는 각 단계마다 "다음 단계로 넘어가게 만드는 장치"를 심어야 합니다. 이 문서 전체가 그 장치를 만드는 방법입니다.

## 2. 우리 제품 한 문장으로 이해하기

**베로로**를 한 문장으로 설명하면:

> **"스마트폰으로 사료 성분표를 찍으면, 유해·알레르기 성분을 바로 정리해주고,
> 광고 없는 진짜 리뷰까지 볼 수 있는 반려동물 식단 앱"**

### 우리가 가진 3가지 무기 (제품 핵심 기능)

| 기능 | 고객에게 주는 가치 | 광고에서 쓸 한 마디 |
|---|---|---|
| 📷 **성분 스캔 (OCR)** | 뒷면 작은 글씨를 대신 읽고 위험 성분을 표시 | "찰칵 한 번이면 성분 분석 끝" |
| ⭐ **광고 없는 찐 리뷰** | 광고성 후기가 아닌 실사용자의 솔직한 평가 | "광고 없는 청정 구역, 진짜 후기만" |
| 🛒 **맞춤 커머스** | 우리 아이에게 맞는 제품 추천 + 앱 안에서 바로 구매 | "우리 아이에게 맞는 것만 골라드려요" |

### 경쟁사 대비 우리만의 강점 (차별점)
- 다른 쇼핑몰: 광고성 리뷰가 많음 → **우리는 광고를 배제한 신뢰 리뷰**
- 성분표: 전문용어라 어려움 → **우리는 카메라로 찍으면 자동 해석**
- 알레르기/노령견 특별 관리 → **맞춤 리포트로 지원 (프리미엄)**

> 📌 **왜 이걸 먼저 정리하나요?** 앞으로 만드는 모든 광고 문구·영상·글은 위 표에서 출발합니다.
> "무슨 말을 하지?" 막힐 때마다 이 표로 돌아오세요.

## 3. 누구에게 팔 것인가 — 타깃 고객(페르소나)

**페르소나** = 우리 고객을 실제 사람처럼 구체적으로 그려본 가상의 인물입니다.
"20~40대 여성" 같은 막연한 대상이 아니라, 아래처럼 **얼굴이 그려질 만큼 구체적으로** 정해야 광고 문구가 잘 나옵니다.

### 페르소나 A — "깐깐한 초보 집사" (1순위 타깃)
- **나이/상황:** 28~35세, 첫 반려동물을 키운 지 1년 이내
- **고민:** "사료 뒷면 성분표를 봐도 무슨 뜻인지 모르겠다. 이게 우리 아이한테 좋은 건가?"
- **평소 행동:** 인스타그램·유튜브에서 반려동물 정보 검색, 후기를 꼼꼼히 읽음
- **우리가 줄 것:** 성분 스캔으로 **불안을 3초 만에 해소**
- **꽂히는 한마디:** "뒷면의 작은 글씨, 매번 불안했죠? 이제 찍기만 하세요."

### 페르소나 B — "알레르기·노령견 보호자" (2순위, 결제 전환 높음)
- **상황:** 아이가 알레르기가 있거나 나이가 많아 식단 관리가 필수
- **고민:** "닭·곡물·유제품 같은 트리거를 매번 확인하기 너무 번거롭다"
- **우리가 줄 것:** 알레르기 성분 자동 표시 + **월간 맞춤 리포트(프리미엄)**
- **꽂히는 한마디:** "우리 아이 트리거, 이제 놓치지 마세요."

### 페르소나 C — "정보 탐색형 실속 소비자"
- **상황:** 광고성 후기에 지쳐 진짜 정보를 찾는 사람
- **우리가 줄 것:** 광고 없는 찐 리뷰
- **꽂히는 한마디:** "광고 아닌 진짜 후기, 여기 다 모았어요."

> ✅ **오늘 할 일:** 위 3명 중 **1명(페르소나 A)** 을 골라 이번 달 광고의 주인공으로 삼으세요.
> 처음에는 대상을 좁힐수록 메시지가 날카로워지고 성과가 좋습니다.

## 4. 무엇을 말할 것인가 — 핵심 메시지

### 4-1. 메시지 만드는 공식 (외워두면 평생 씁니다)

```
[고객의 고민] → [우리 해결책] → [행동 요청(CTA)]
```

**예시:**
> "성분표 봐도 모르겠어서 불안하셨죠? (고민)
> 베로로는 찍기만 하면 위험 성분을 정리해드려요. (해결)
> 지금 무료로 스캔해보세요. (행동)"

- **CTA(Call To Action)** = "행동 요청". 고객에게 시킬 다음 행동을 **딱 하나** 정해서 명확히 말하는 것.
  - 좋은 예: "앱 다운로드하기", "무료로 성분 스캔하기", "사전 예약하고 혜택받기"
  - 나쁜 예: (CTA 없음) 또는 "구경하세요" 처럼 애매한 표현

### 4-2. 상황별 핵심 슬로건 (그대로 사용 가능)

| 상황 | 슬로건 |
|---|---|
| 대표 슬로건 | **"우리 아이 입에 들어가는 것, 정말 안전할까요?"** |
| 성분 기능 강조 | "찰칵 한 번으로 끝나는 성분 분석" |
| 리뷰 기능 강조 | "광고 없는 청정 구역, 찐 리뷰" |
| 프리미엄 강조 | "월간 맞춤 리포트로 우리 아이 식단을 설계하세요" |
| 짧은 광고용 | "성분표, 이제 찍기만 하세요" |

### 4-3. 메시지 톤 앤 매너 (말투 규칙)
- **친근하고 다정하게.** 보호자의 마음을 이해하는 말투 ("~하셨죠?", "마음 졸이셨던")
- **불안을 자극하되 협박하지 않기.** 겁주는 게 아니라 **안심**을 파는 겁니다.
- **과장 금지.** "무조건 안전" 같은 표현은 신뢰를 깎습니다. "확인할 수 있어요"가 맞습니다.
- 이모지는 1~2개만. 너무 많으면 신뢰가 떨어집니다.

## 5. 어디서 알릴 것인가 — 채널별 실행 방법

초보자는 **채널을 다 하려다 다 망합니다.** 아래에서 **처음엔 ①~③만** 집중하세요.

### ① 인스타그램 (1순위 — 여기부터 시작)
반려동물 콘텐츠가 가장 잘 퍼지는 곳입니다.

**세팅 (한 번만):**
1. 비즈니스 계정으로 전환 (설정 → 계정 유형 → 프로페셔널 계정)
2. 프로필에 한 줄 소개 + 앱 다운로드 링크 넣기
3. 프로필 사진은 로고(`landing/public/veroro-logo.png` 활용)

**매주 반복할 일:**
- **릴스(짧은 영상) 주 3개** — 성분 스캔하는 15초 영상이 가장 반응 좋음
- **피드 게시물 주 2개** — "이 성분 아세요?" 정보 카드
- **스토리 매일 1개** — 투표, 질문 스티커로 소통

**해시태그 (게시물마다 10~15개):**
```
#반려견 #강아지사료 #고양이사료 #반려묘 #사료추천 #강아지간식
#반려동물 #댕댕이 #냥스타그램 #멍스타그램 #사료성분 #반려견건강
#펫푸드 #강아지알레르기 #베로로
```

### ② 유튜브 쇼츠 / 틱톡 (2순위)
- 인스타 릴스에 올린 영상을 **그대로 재활용**해서 올리면 됩니다 (재촬영 X).
- 제목 앞부분에 **후킹 문구** 필수: "사료 뒷면, 이거 모르면 큰일 나요"

### ③ 네이버 블로그 & 검색(SEO) (3순위 — 오래 가는 자산)
사람들은 네이버에 "강아지 사료 추천", "고양이 알레르기 성분" 을 검색합니다.
여기서 우리 글이 뜨면 광고비 없이 계속 고객이 들어옵니다.

- **주 1~2회 블로그 글** 작성. 주제 예:
  - "강아지 사료 성분표 읽는 법 (초보 집사 필독)"
  - "우리 강아지 알레르기, 어떤 성분을 피해야 할까?"
  - "광고 아닌 진짜 사료 후기 찾는 법"
- 글 끝에 **항상 앱 다운로드 링크** 넣기.
- **SEO(검색 최적화)** = 검색에 잘 걸리게 만드는 것. 제목·본문에 사람들이 검색하는 단어(키워드)를 자연스럽게 넣으면 됩니다.

### ④ 커뮤니티 마케팅 (돈 안 들고 효과 좋음)
반려동물 카페·오픈채팅·앱 커뮤니티에 **광고 티 안 나게** 도움을 주며 자연스럽게 소개.
> ⚠️ 주의: 대놓고 광고하면 강퇴당합니다. **진짜 정보로 도움 → 필요할 때만 언급**이 원칙.

### ⑤ 앱스토어 최적화 (ASO)
- **ASO(App Store Optimization)** = 앱스토어 검색에서 잘 뜨게 만드는 것.
- 앱 제목·설명에 "사료 성분 분석", "반려동물 리뷰" 같은 검색어 포함
- 스크린샷 첫 장에 대표 슬로건 크게
- **초기 리뷰·별점**이 매우 중요 → 지인·초기 사용자에게 정중히 리뷰 부탁

### ⑥ 인플루언서 (예산 생기면)
- **처음엔 팔로워 1만~5만의 반려동물 마이크로 인플루언서**가 가성비 최고.
- 무료 프리미엄 제공 + 소정의 비용으로 협업.
- "성분 스캔 실사용 후기" 콘텐츠를 요청 (제품이 자연스럽게 녹아야 함).

### ⑦ 유료 광고 (기초가 잡힌 뒤)
- 인스타/페이스북 광고(Meta 광고 관리자)부터. **하루 1만~2만 원**으로 소액 테스트.
- 처음엔 "앱 설치" 목표로 설정, 페르소나 A 대상으로.

> ✅ **채널 우선순위 요약:** 처음 한 달은 **인스타그램 + 쇼츠 + 블로그** 3개만.
> 익숙해지면 커뮤니티·ASO 추가, 성과가 보이면 인플루언서·유료광고로 확장.

## 6. 4주 런칭 플랜 (그대로 따라 하기)

하루 1~2시간 기준으로 짠 현실적인 계획입니다.

### 1주차 — 준비 (기초 다지기)
- [ ] 이 문서 3·4장을 읽고 **페르소나 1명 + 슬로건 1개** 확정
- [ ] 인스타그램 비즈니스 계정 생성 + 프로필 세팅
- [ ] 유튜브 채널 + 네이버 블로그 개설
- [ ] 콘텐츠 소재 30개 목록 작성 (7장 참고)
- [ ] 촬영 도구 준비 (스마트폰이면 충분)

### 2주차 — 콘텐츠 만들기 (창고 채우기)
- [ ] 성분 스캔 시연 영상 5개 촬영·편집
- [ ] 정보 카드 게시물 5개 제작 (Canva 등 무료 툴 사용)
- [ ] 블로그 글 2개 작성
- [ ] 사전 예약/다운로드 링크가 모든 콘텐츠에 들어갔는지 확인

### 3주차 — 발행 & 소통 (세상에 알리기)
- [ ] 릴스/쇼츠 매일 1개씩 발행 시작
- [ ] 댓글·DM에 24시간 안에 답하기 (초기엔 소통이 알고리즘에 매우 유리)

### 4주차 — 측정 & 개선 (숫자 보고 방향 잡기)

## 8. 예산 짜기 — 돈이 적을 때 vs 있을 때

### 💸 예산 0원 (시간만 투자)
- 인스타/쇼츠/블로그 직접 운영
- 커뮤니티 활동
- 지인·초기 사용자 리뷰 확보
- **가장 중요:** 꾸준함. 하루 1시간씩 3개월이면 유료광고보다 강합니다.

### 💸 월 30만 원 수준
| 항목 | 금액 | 설명 |
|---|---|---|
| 인스타/Meta 유료광고 | 15만 원 | 하루 5천 원, 앱 설치 목표 |
| 마이크로 인플루언서 1명 | 10만 원 | 제품+소정 비용 협업 |
| 디자인 툴(Canva Pro 등) | 2만 원 | 콘텐츠 퀄리티 |
| 예비비 | 3만 원 | 반응 좋은 광고 추가 집행 |

### 💸 월 100만 원 이상
- 위 항목 확대 + 인플루언서 여러 명 + 유료광고 20만 원 이상으로 A/B 테스트
- **A/B 테스트** = 광고 문구·이미지 2개를 동시에 돌려 더 잘 되는 걸 남기는 방법

> ✅ **예산 원칙:** 처음엔 무조건 **소액으로 테스트 → 잘 되는 것에만 돈을 더** 넣으세요.
> "잘 될 것 같아서" 큰돈을 한 번에 쓰는 게 초보자의 가장 큰 실수입니다.

## 9. 숫자로 확인하기 — KPI와 측정 방법

**KPI(Key Performance Indicator)** = 잘 되고 있는지 판단하는 핵심 숫자.
감이 아니라 **숫자로** 판단해야 합니다.

### 단계별로 봐야 할 숫자
| 퍼널 단계 | 핵심 지표 | 어디서 보나 | 목표(예시) |
|---|---|---|---|
| 인지 | 도달수/조회수 | 인스타 인사이트, 유튜브 스튜디오 | 월 5만 |
| 관심 | 프로필 방문, 링크 클릭 | 인스타 인사이트 | 조회의 3%+ |
| 설치 | 앱 다운로드 수 | 앱스토어/구글플레이 콘솔 | 월 500 |
| 활성 | 성분 스캔 실행 수 | 앱 내 분석 데이터 | 설치의 60%+ |
| 결제 | 프리미엄 구독/구매 | 앱 결제 데이터 | 활성의 3~5% |
| 추천 | 재방문·추천 수 | 앱 데이터 | 지속 상승 |

### 매주 5분 점검 루틴
1. 위 표에서 지난주 숫자 기록
2. **오른 지표 / 내린 지표** 표시
3. 오른 건 "무엇 때문에?" → 더 하기
4. 내린 건 "무엇 때문에?" → 원인 메모 후 실험

> 💡 초보자는 지표를 **딱 3개만** 집중해도 됩니다: **① 조회수 ② 앱 설치 수 ③ 성분 스캔 실행 수.**

### 측정 도구 (무료)
- 인스타그램 인사이트 (앱 내장)
- 유튜브 스튜디오
- 네이버 블로그 통계
- 구글 애널리틱스(GA4) — 랜딩페이지(`landing/`) 방문 분석용
- 앱스토어 커넥트 / 구글 플레이 콘솔 — 설치·리뷰 확인

## 11. 초보자가 자주 하는 실수 10가지

1. **채널을 한꺼번에 다 하려다 지침** → 처음엔 3개만.
2. **완벽한 콘텐츠만 올리려다 못 올림** → 일단 발행, 반응 보고 개선.
3. **"우리 앱 좋아요"만 반복** → 고객의 고민부터 말하기.
4. **CTA(행동 요청)가 없음** → 모든 콘텐츠 끝에 "다음에 뭘 하라"를 넣기.
5. **숫자를 안 봄** → 감이 아니라 KPI로 판단.
6. **큰돈을 한 번에 광고에 씀** → 소액 테스트 후 확대.
7. **댓글·DM 방치** → 초기엔 소통이 최고의 마케팅.
8. **타깃이 너무 넓음** → 페르소나 1명으로 좁히기.
9. **과장·허위 광고** → 신뢰 붕괴. "확인할 수 있다"까지만.
10. **한 달 하고 포기** → 마케팅은 최소 3개월 누적이 필요.

## 12. 용어 사전

| 용어 | 쉬운 설명 |
|---|---|
| 타깃/페르소나 | 우리 고객을 실제 사람처럼 구체화한 것 |
| 포지셔닝 | 고객 머릿속에 "이 브랜드는 ○○" 라고 각인시키는 것 |
| 퍼널(Funnel) | 인지→관심→설치→사용→결제로 이어지는 고객 여정 |
| CTA | Call To Action, "다음에 이걸 하세요"라는 행동 요청 |
| CTR | Click Through Rate, 본 사람 중 클릭한 비율 |
| 전환(Conversion) | 원하는 행동(설치·결제 등)을 실제로 한 것 |
| CVR | Conversion Rate, 전환율 |
| ROAS | 광고비 대비 매출. 1만 원 써서 3만 원 벌면 ROAS 300% |
| CAC | 고객 1명 데려오는 데 든 비용 |
| SEO | 검색에 잘 걸리게 글·페이지를 최적화하는 것 |
| ASO | 앱스토어 검색에 잘 걸리게 최적화하는 것 |
| A/B 테스트 | 두 버전을 동시에 돌려 더 나은 걸 고르는 실험 |
| 도달(Reach) | 콘텐츠를 본 사람 수 |
| 인게이지먼트 | 좋아요·댓글·공유 등 반응 |
| KPI | 성과를 판단하는 핵심 숫자 |
| 오가닉/페이드 | 무료(자연) 유입 / 유료 광고 유입 |
